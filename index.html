终于搭建好自己的博客了

记录一下用hexo搭建博客入的坑

一开始还是很顺利的，直到 hexo deploy 部署这一步.点下命令之后没有反应，于是开始google.
查到hexo的官方github里有类似问题，解决方案竟然是在项目根目录的_config.yml的deploy标签下的所有标签:后都要加”空格”.

联想到 其实整个_config.yml文件的格式都是如此 :后要加”空格”

之后再hexo deploy的时候可能会报
ERROR Deployer not found: git 或者 ERROR Deployer not found: github

解决方法： npm install hexo-deployer-git –save

然后关于github的关联 只要注意自己的账号有没有认证email就行了。
